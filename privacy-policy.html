Tokenizer — Privacy Policy

Last updated: March 12, 2026  ·  Version 0.5.2
🔒 Short version: Tokenizer collects zero data. Everything runs locally in your browser. Nothing leaves your device.

1. What Data We Collect

None. Tokenizer does not collect, store, transmit, or share any personal data, browsing data, message content, or usage statistics.

2. How Token Counting Works

All token counting and cost estimation runs entirely within your browser using local JavaScript. The text you type is never read by Tokenizer's code outside of the browser tab you're currently viewing, and is never sent to any server — not ours, not anyone's.

3. Local Storage

Tokenizer may use your browser's local storage API to save your session token count and preferences (such as overlay position) locally on your device. This data never leaves your device and is not accessible to us or any third party.

4. API Interception

Tokenizer intercepts outgoing API responses on supported platforms to read token usage metadata (token counts returned by the AI provider in their API responses). This interception happens locally in your browser. Tokenizer does not modify, block, or retransmit any requests or responses. Your API keys are never read or stored.

5. Third-Party Services

Tokenizer does not use any analytics, advertising, tracking, or crash reporting services. There are no third-party SDKs embedded in the extension.

6. Open Source

Tokenizer is fully open source. You can review every line of code at:
https://github.com/gyeningcorp/tokenizer

7. Children's Privacy

Tokenizer does not knowingly collect data from anyone, including children under 13.

8. Changes

If we ever change this policy (which would only occur if we added features that process data), we will update this page and bump the version number above.
